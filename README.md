👁️‍🗨️ Object Detection for Blind People

An AI-powered mobile application designed to assist visually impaired individuals by providing real-time object detection and audio feedback.

🚀 Overview

This project aims to improve the independence and safety of visually impaired users by using Artificial Intelligence and Computer Vision. The system detects objects, reads text, recognizes currency, and provides voice-based interaction through a mobile application.

✨ Features
Real-time Object Detection using YOLO-based models
Text Recognition (OCR for signboards and documents)
Currency Detection
Voice Command Integration
Text-to-Speech (Audio Feedback)
AI Assistant using Gemini API
Navigation Support (GPS-based)
Document Understanding
🏗️ System Architecture

The system consists of a Flutter-based frontend and a Python-based backend. The frontend handles user interaction, camera input, and voice commands, while the backend processes data using AI models such as YOLO and OCR. External APIs like Google Gemini are used for voice and AI interaction. The system works in real time and does not rely on database storage.

⚙️ How It Works

The user provides input through the camera or voice command. This input is sent to the backend where AI models process it for object detection, text recognition, or other tasks. The results are then converted into audio using text-to-speech and delivered to the user in real time.

🛠️ Tech Stack
Languages: Python, Dart
Framework: Flutter
Libraries: OpenCV, EasyOCR
Models: YOLOv8, TensorFlow Lite
APIs: Google Gemini API
⚙️ Installation

Clone the repository from GitHub and navigate to the project folder. Install the required dependencies using the requirements file. Run the backend using Python and then start the Flutter application using standard Flutter commands.


🔮 Future Scope

The system can be further improved by integrating wearable devices such as smart glasses, enhancing outdoor navigation features, improving model accuracy, and expanding multi-language support.
